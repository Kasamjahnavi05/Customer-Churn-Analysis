###  Insights From My Dashboard

| Area                     | What I Found                                                            |
| ------------------------ | ----------------------------------------------------------------------- |
| **Demographics**         | Senior citizens and low-tenure customers have higher churn.             |
| **Service Subscription** | Fiber optic users churn more; customers with fewer services churn less. |
| **Contracts**            | Month-to-month contracts show the highest churn rate.                   |

| **Key Drivers**          | Contract type, internet service, and tenure are major influencers.      |
